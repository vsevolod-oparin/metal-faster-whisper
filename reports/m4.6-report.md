# M4.6 Milestone Report — Main Decode Loop

**Date:** 2026-03-19
**Status:** PASSED (5/5 tests)

## Summary

Implemented the complete `transcribe` pipeline — the top-level method that takes an audio file or samples and returns timed, segmented transcription text. This ties together all M0-M4.5 components into a working end-to-end Whisper transcription system.

## Changes

- **MWTranscriber.h** — added `MWTranscriptionSegment` (id, seek, start, end, text, tokens, temperature, avgLogProb, compressionRatio, noSpeechProb), `MWTranscriptionInfo` (language, languageProbability, duration), `transcribeURL:` and `transcribeAudio:` methods, `MWErrorCodeTranscribeFailed`
- **MWTranscriber.mm** — ~200 lines added:
  - `_modelPath` ivar for creating task/language-specific tokenizers
  - `sliceMel()` helper for extracting frame ranges from full mel spectrogram
  - Option parsing helpers for the NSDictionary options
  - `transcribeURL:` — audio decode → delegate to `transcribeAudio:`
  - `transcribeAudio:` — the main decode loop:
    1. Compute mel spectrogram for entire audio
    2. Auto-detect language (or use provided)
    3. Create tokenizer for detected language + task
    4. Build suppressed tokens
    5. Handle initial_prompt
    6. Parse clip_timestamps → seek_clips
    7. Main loop: extract mel → pad/trim → encode → build prompt → generate with fallback → no-speech check → split by timestamps → yield segments
    8. Condition on previous text with prompt reset on temperature
    9. Infinite loop protection (force-advance seek)

## Pipeline Flow

```
Audio File/Samples
  → MWAudioDecoder (16kHz mono float32)
  → MWFeatureExtractor (mel spectrogram)
  → Language detection (optional)
  → For each 30s chunk:
      → Slice mel → Pad/trim to 3000 frames
      → Encode (CT2 encoder, Metal GPU)
      → Build prompt (previous context, hotwords)
      → Generate with temperature fallback (CT2 decoder, Metal GPU)
      → No-speech detection → skip if silent
      → Split by timestamp tokens → MWSegmentInfo
      → Decode text → MWTranscriptionSegment
      → Call segmentHandler callback (streaming)
  → Return all segments + MWTranscriptionInfo
```
