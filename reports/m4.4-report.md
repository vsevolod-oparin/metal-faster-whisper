## Temperature Fallback Logic

For each temperature in the list:
1. **temp=0**: beam search with `beam_size`, `patience`
2. **temp>0**: sampling with `beam_size=1`, `num_hypotheses=bestOf`, `sampling_topk=0`
3. Compute `avgLogProb = (score √ó seqLen^lengthPenalty) / (seqLen + 1)`
4. Decode text, compute compression ratio
5. Check fallback conditions:
   - `compressionRatio > threshold` ‚Üí fallback (too repetitive)
   - `avgLogProb < threshold` ‚Üí fallback (low confidence)
   - `noSpeechProb > threshold AND avgLogProb < logProbThreshold` ‚Üí NO fallback (silence)
6. If all temps fail: select best result (highest avgLogProb, prefer below-CR-threshold)
